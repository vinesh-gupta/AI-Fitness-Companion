# AI Fitness Companion

## Overview
AI Fitness Companion is an AI-powered fitness assistant that helps users improve their fitness journey through personalized workout planning, diet recommendations, habit tracking, and behavioral analysis.

## Features

### AI Dietician
- Calculates BMI
- Generates personalized diet recommendations
- Uses Google Gemini AI

### Virtual Gym Buddy
- Answers fitness-related questions
- Provides workout guidance and exercise suggestions

### Workout Planner
- Creates workout plans based on fitness goals and experience level

### Habit Tracker
- Tracks completed workouts
- Stores workout history using SQLite

### Behavioral AI
- Analyzes workout consistency
- Predicts risk of missing future workouts
- Provides motivational nudges

### Admin Dashboard
- Displays workout analytics
- Shows engagement score
- Monitors user activity

## Technologies Used
- Python
- Streamlit
- SQLite
- Google Gemini API
- Pandas

## Installation

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
